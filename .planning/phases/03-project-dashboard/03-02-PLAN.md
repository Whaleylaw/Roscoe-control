**Task metrics (reactive via Zustand, per D-04, D-18):**
```typescript
const { tasks } = useMissionControl()
const { project, slug } = useProjectWorkspace()

const projectTasks = useMemo(() =>
  tasks.filter(t => t.project_id === project?.id),
  [tasks, project?.id]
)

const STATUS_GROUPS = {
  active: ['inbox', 'assigned', 'awaiting_owner', 'in_progress', 'review', 'quality_review'],
  blocked: ['failed'],
  completed: ['done'],
} as const

const counts = useMemo(() => {
  const active = projectTasks.filter(t => STATUS_GROUPS.active.includes(t.status as any)).length
  const blocked = projectTasks.filter(t => STATUS_GROUPS.blocked.includes(t.status as any)).length
  const completed = projectTasks.filter(t => STATUS_GROUPS.completed.includes(t.status as any)).length
  return { active, blocked, completed, total: active + blocked + completed }
}, [projectTasks])
```
Note: `total` excludes `backlog` tasks per research recommendation (Open Question 2). This prevents a project with 100 backlog items showing 2% progress.

**Activity feed (API fetch + SSE refresh, per D-10, D-19):**
```typescript
const [activities, setActivities] = useState<Activity[]>([])

const fetchActivities = useCallback(async () => {
  if (!project?.id) return
  try {
    const res = await fetch('/api/activities?entity_type=task&limit=50')
    const data = await res.json()
    const projectTaskIds = new Set(projectTasks.map(t => t.id))
    const filtered = (data.activities || [])
      .filter((a: Activity) => a.entity_type === 'task' && projectTaskIds.has(a.entity_id))
      .slice(0, 20)
    setActivities(filtered)
  } catch { /* silently fail, show empty state */ }
}, [project?.id, projectTasks])

useEffect(() => { fetchActivities() }, [fetchActivities])
```
Per D-17/D-18: The task counts update automatically because `useMissionControl().tasks` is updated by SSE globally. For activities, re-fetch when `projectTasks` changes (the dependency in `useCallback` handles this). This means when SSE updates a task in the store, `projectTasks` recomputes, `fetchActivities` gets a new reference, and the `useEffect` re-runs to refresh activities.
