<objective>
Ship the heaviest Phase 14 endpoint: `POST /api/runner/claim/[task_id]`. A single request must atomically claim the task, re-validate mounts + caps against allowlist, enforce concurrency caps, mint a per-attempt runner-token, insert a task_runner_attempts history row, and return a full dispatch payload the daemon can hand to `docker run`.

Purpose: Claim is the critical section where all of Phase 11/12/13's substrate comes together. One handler ties runner-secret auth → recipe lookup → allowlist validation → concurrency caps → atomic status flip → token mint → prior-attempts history — each of which is a pre-shipped module. Pure helpers are factored out into `src/lib/runner-claim.ts` so they can be unit-tested without the HTTP layer.
Output: One route + one helper module + two test suites covering happy + cap + allowlist + double-claim cases.
</objective>

max_attempts filesystem re-parse helper (LOCKED in locked_decisions; getIndexedRecipeBySlug verified to NOT carry max_attempts):
```typescript
// src/lib/runner-claim.ts
import { getRecipesRoot } from '@/lib/recipe-watcher'
import { parseRecipeYaml } from '@/lib/recipe-schema'
import { readFileSync, existsSync } from 'node:fs'
import { join } from 'node:path'

export function resolveRecipeMaxAttempts(slug: string): number | undefined {
  const recipesRoot = getRecipesRoot()
  const yamlPath = join(recipesRoot, slug, 'recipe.yaml')
  if (!existsSync(yamlPath)) return undefined
  try {
    const raw = readFileSync(yamlPath, 'utf8')
    const parsed = parseRecipeYaml(raw)
    if (parsed.ok) return parsed.value.max_attempts
  } catch { /* fall through */ }
  return undefined
}
```

      const result = db.prepare(`
        UPDATE tasks
        SET status = 'in_progress',
            container_id = ?,
            runner_started_at = ?,
            runner_attempts = runner_attempts + 1,
            updated_at = ?
        WHERE id = ?

          AND container_id IS NULL
          AND recipe_slug IS NOT NULL
      `).run(placeholder, nowUnix, nowUnix, taskId)
      if (result.changes === 0) return { claimed: false as const }

        INSERT INTO task_runner_attempts (task_id, attempt, started_at)
