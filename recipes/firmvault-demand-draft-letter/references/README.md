# Draft Demand Reference

This recipe drafts the demand package after demand readiness has been reviewed and the internal final-lien process check has run.

Expected canonical write targets:

- `demand/demand-letter.md`
- `demand/damages-summary.md`
- `demand/demand-package.md`

Expected evidence sources:

- `demand/readiness.md`
- `accident/accident.md`
- `accident/liability.md`
- `insurance/bi-*.md`
- `insurance/um-*.md`
- `insurance/uim-*.md`
- `medical-providers/<provider>/chronology.md`
- `medical-providers/<provider>/records-bills.md`
- `medical-providers/<provider>/documents/bills.md`

Lien facts are internal case-management facts. Do not include lien information in demand content, and do not add a note saying liens were excluded.
