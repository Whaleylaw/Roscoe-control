# Review: Medical Records Request Preparation

Approve only if the request preparation is specific, evidence-backed, and ready for human sending or already documented as prepared.

Check:

- Authorization was confirmed first.
- Providers and treatment date ranges came from the masked vault.
- The request asks for both records and itemized bills unless a supported reason says otherwise.
- Any missing data is surfaced as a blocker.
- No external sending occurred.
- File changes are limited to the assigned case.

Reject if provider facts were invented, raw PHI was added, the request is too vague to send, or unrelated files changed.
