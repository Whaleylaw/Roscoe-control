# Review: Settlement Lien Result Documentation

Approve only if the worker documented the human-reviewed lien outcome without crossing into payment or final distribution.

The worker must:

- read same-task comments and `liens/settlement-lien-audit.md`,
- update lien/distribution planning only from supported facts,
- document no-lien clearance when applicable,
- preserve unresolved lien/payment blockers,
- append activity and workflow-log entries.

Reject if the worker invents reductions or payment facts, marks liens paid without proof, issues distribution, marks final distribution complete, or ignores a human-reported missing lien.
