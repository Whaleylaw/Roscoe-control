# Review: Authorization to Settle Draft

Approve only if the worker prepared an authorization package for human signature without claiming signature or external contact.

The worker must:

- rely on the draft settlement statement,
- write `documents/generated/settlement/authorization-to-settle-draft.md`,
- write a human signature handoff,
- update settlement status only to authorization prepared,
- append activity and workflow-log entries.

Reject if the worker claims the client signed, contacts the client, marks release/funds/distribution milestones, invents final financial numbers, or edits old logs.
