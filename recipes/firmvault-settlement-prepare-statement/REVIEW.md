# Review: Settlement Statement Draft

Approve only if the worker prepared a supported draft settlement statement and preserved uncertainty instead of inventing financial facts.

The worker must:

- use canonical accepted-settlement evidence,
- write `documents/generated/settlement/settlement-statement-draft.md`,
- update `settlement/distribution.md` as draft planning only,
- clearly label missing fee, cost, lien, trust, or net values as TBD/needs review,
- append new activity and workflow-log entries.

Reject if the worker invents numbers, marks funds received, pays liens, issues checks, treats estimates as final, edits old logs, or creates a final distribution.
