# Review: Second Medical Records Follow-Up

Approve only if the worker checked post-first-follow-up status and documented a specific next action.

Check:

- Prior request and first follow-up are identified.
- Pending reason is specific where available.
- The handoff is actionable and provider-specific.
- No unsupported receipt or request status was created.
- No external communication was performed.

Reject if the worker duplicates the first follow-up without checking current status or fabricates provider responses.
