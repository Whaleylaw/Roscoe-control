# Demand Lien-Process Check Reference

This recipe exists because final-lien work can take time and should begin before settlement if a lien is known. It is not demand-letter content.

Expected canonical write target:

- `liens/final-lien-readiness.md`

Expected evidence sources:

- `liens/*.md`
- `demand/readiness.md`
- `activity/`
- `workflow-log/`

If final-lien work has not started, the output should make the next workflow obvious without saying the final amount was requested.
